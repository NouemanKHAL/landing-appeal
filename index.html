I'm not a developper, is Wooble for me ?

It is for you! Wooble aims to help people to build beautiful websites with very little development knowledge. Actually, using Wooble requires to know what a script tag is in HTML. Be sure that Wooble will support you, Wooble loves adventurers who are eager to build!

Is Wooble free ?

For little use Wooble is totally free. Wooble works with a packages system that is like folders where you put in the creations you want, one package can work for only one website. With a free account you have one package in which you can put at most five creations.

Why Wooble ? I can use templates if I want to create a beautiful website.

That is right and I encourage you to use templates when needed. Wooble don't replace templates, it's not our mission. In fact, it turns out to be better than replacing, Wooble can be combined with templates ! Yup ! Have you ever used a template where everything looks cool but the call to action button is not satisfying ? Well, search on Wooble, find this one call to action button that will make your landing page to the top, package it and replace the not-so-good button by the new ultimate button from Wooble.
