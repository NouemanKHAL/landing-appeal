What is Wooble?

Wooble is a website in which web designers build and publish their creations. Wooble is also an infrastructure that allows everybody to easily implement the published creations in websites.

I'm a web designer, why should I create on Wooble?

Users come in Wooble to see what they can use for their website, that means your everybody will recognized your work. Wooble is an opportunity to meet people who might be interested in working with you in the future. People always need a web designer to collaborate with.

I'm not a developper, is Wooble for me?

It is for you! Wooble aims to help people to build beautiful websites with very little development knowledge. Actually, using Wooble requires to know what a script tag is in HTML. Be sure that Wooble will support you, Wooble loves adventurers who are eager to build!

Technically, how Wooble works?

Wooble has a packages system, a package is a container in which users can put every published creations they want. Then, Wooble compile every creations inside the package and returns to the user a URL that links to the compiled package script. The script is like an interface that allows to pick one creation at a time and do everything possible with it.
